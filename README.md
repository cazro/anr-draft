# ANR Draft
Now for Discord

## Install

```python setup.py install```

Or just...

```pip install discord```

since that's the only dependency.

## Setup

Put your bot's token in anrdraft/secrets.json

There is an example file that you can use and rename to secrets.json

## Actions
#### Create Draft

!createdraft

#### Join Draft

!joindraft [draft ID]

#### Begin Draft

!startdraft [draft ID]

#### Pick Card

!pick [card code]

#### Leave Draft

!leavedraft [draft ID]
